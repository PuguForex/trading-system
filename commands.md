# Commands Reference

## Project Setup

npm init -y
# Initializes Node project

npm install typescript --save-dev
# Installs TypeScript

npx tsc --init
# Creates tsconfig.json

## Compilation

npx tsc
# Compiles TypeScript to JavaScript

node dist/index.js
# Runs compiled code

## Git

git init
# Initialize repository

git add .
# Stage changes

git commit -m "message"
# Save snapshot
